NexusFlow is a comprehensive B2B SaaS platform designed for modern enterprises. It provides seamless management for analytics, team collaboration, and scheduling, wrapped in a highly interactive and elegant user interface.

🌐 **Live Demo:** [Link Vercel Anda di sini]

---

## 🎓 Academic Context
This project was developed as the **Alternative Objective Learning (AOL)** for the **Software Engineering** course at **BINUS University** (Semester 4). 

Built by a team of Computer Science students specializing in **Database Systems**, the application emphasizes a robust and secure data architecture utilizing Multi-Tenant PostgreSQL with Row Level Security (RLS).

---

## ✨ Key Features

* **📊 Dynamic Dashboard:** Real-time KPI tracking, interactive revenue growth charts, and traffic source analytics.
* **👥 Team Management:** Manage members, roles, and permissions securely.
* **📅 Interactive Calendar:** Dynamic event scheduling, real-time updates, and conflict management.
* **🔐 Secure Authentication:** Enterprise-grade login and session management powered by Supabase Auth.
* **✨ Modern UI/UX:** Smooth animations (Framer Motion), responsive design (Tailwind CSS), and accessible components.
